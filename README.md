📋 Registration Form with Validation

This project is a simple Registration Form built using HTML, CSS, and JavaScript. It performs real-time validation for user inputs and ensures data correctness before enabling submission.

🚀 Features
✅ Name field validation (cannot be empty)
✅ Email validation using regex pattern
✅ Password validation (minimum 6 characters)
✅ Real-time input validation
✅ Submit button enabled only when all fields are valid
✅ Simple and clean UI design

🛠️ Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS)

📂 Project Structure
project-folder/
│── index.html

⚙️ How It Works
The form listens to user input using addEventListener
Each field is validated dynamically:
Name → must not be empty
Email → must match email format
Password → minimum 6 characters
If all validations pass:
✔ Submit button becomes enabled
Otherwise:
❌ Error messages are shown
▶️ How to Run
Download or clone the repository
Open the index.html file in any browser
